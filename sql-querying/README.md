# SQL Querying Exercise

Solutions for the four-part SQL Querying exercise from Unit 17.2.

## Files

- `crud.sql` — Database/table setup for Part 3 (products).
- `queries_products.sql` — Final SQL for the 14 products tasks (Part 3).
- `queries_playstore.sql` — Final SQL for the 15 Google Play Store tasks plus the four bonus queries (Part 4 + Further Study).
- `sqlzoo_codewars.md` — Answers to the Part 1 SQL Zoo tutorials and Part 2 Codewars kata.

## Running

```bash
psql < seed_products.sql
psql < seed_playstore.sql
```

Then run the queries in `queries_products.sql` and `queries_playstore.sql` against the resulting databases.
